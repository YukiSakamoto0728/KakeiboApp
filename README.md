# 家計簿アプリ

## 概要

C# WinForms と SQLite を使用して作成した家計簿アプリです。


収入・支出の管理、データ保存、グラフ表示を行えます。

## セットアップ

1. リポジトリをクローンまたはダウンロード
2. Visual Studioでソリューションファイル(.sln)を開く
3. NuGetパッケージを復元
4. 実行(F5)


  
## 画面

### ホーム画面

収入・支出の合計とグラフを表示

![ホーム画面](images/home.png)


### 収入管理

収入データの登録・削除・編集

削除ボタンを押すと"削除しますか？"と表示された小ウィンドウが表示される。

列をダブルクリックで編集フォームが表示される。

次へ、前へボタンでページの移動ができる。

![収入一覧](images/income.png)

### 支出管理

支出データの登録・削除・編集

削除ボタンを押すと"削除しますか？"と表示された小ウィンドウが表示される。

列をダブルクリックで編集フォームが表示される。

次へ、前へボタンでページの移動ができる。

![支出一覧](images/expense.png)

## 工夫した点

### UserControlを利用して画面切り替えを実装

フォームを増やさずに画面遷移を実現し、保守性を向上させた。

### ページング機能を実装

大量データ表示時の視認性を考慮し、1ページごとにデータを表示できるようにした。

### 削除確認ダイアログを実装

誤操作によるデータ削除を防ぐため、確認メッセージを表示するようにした。

### 円グラフによる可視化

収支状況を直感的に把握できるようにした。
