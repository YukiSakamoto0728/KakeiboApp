# 家計簿アプリ

## 概要

C# WinForms と SQLite を使用して作成した家計簿アプリです。


収入・支出の管理、データ保存、グラフ表示を行えます。

## 反省点

### 棒グラフの縦軸が0から始まっていないため、比率がわからない。

### 収入、支出一覧画面のページが行の幅に対して大きすぎる。

### UI設計を後回しにしたため画面レイアウトの修正が多く発生した。

### コードの共通化を意識していたが、IncomePageとExpensePageで重複コードが多くなった。
